# -# Изгиб Питона — Змейка 🐍

Классическая игра «Змейка», реализованная на Python с использованием принципов ООП.

## О проекте

Простые правила, минималистичный дизайн и драйв — причины популярности этой игры на протяжении многих лет. Пространство игры разделено на ячейки 20×20 пикселей, змейка движется по полю и «съедает» яблоки, увеличиваясь в длине.

## Правила игры

- Управляйте змейкой с помощью **стрелок** на клавиатуре
- Собирайте **красные яблоки**, чтобы увеличивать длину змейки
- Змейка **проходит сквозь стены** (появляется с противоположной стороны поля)
- Игра заканчивается, когда змейка **кусают сама себя** (сброс к начальному состоянию)
- Нажмите **ESC** для выхода из игры

## Технические детали

| Параметр | Значение |
|----------|----------|
| Размер поля | 640×480 пикселей |
| Размер ячейки | 20×20 пикселей |
| Размер сетки | 32×24 ячеек |
| Скорость игры | 20 FPS |
| Цвет змейки | Зелёный (0, 255, 0) |
| Цвет яблока | Красный (255, 0, 0) |
| Фон | Чёрный (0, 0, 0) |

## Классы проекта (ООП)

| Класс | Описание |
|-------|----------|
| `GameObject` | Базовый класс для игровых объектов (позиция, цвет, метод draw) |
| `Snake` | Класс змейки: движение, отрисовка, обработка столкновений, сброс |
| `Apple` | Класс яблока: случайная генерация позиции, отрисовка |

## Установка

1. Установите Python 3.7 или выше
2. Установите зависимости:
   ```bash
   pip install pygame
   ```

## Запуск

```bash
python snake_game.py
```

## Управление

| Клавиша | Действие |
|---------|----------|
| ↑ | Движение вверх |
| ↓ | Движение вниз |
| ← | Движение влево |
| → | Движение вправо |
| ESC | Выход из игры |

## Технологии

- **Python 3.12** — язык программирования
- **Pygame 2.6** — библиотека для создания игр
